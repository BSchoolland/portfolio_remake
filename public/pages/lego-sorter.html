Project: Lego Sorter

I've always loved legos, they come in all shapes and sizes and you can build anything you can imagine! The downside: those shapes and sizes are usually shoved into a single box, so you can't find the pieces you need, meaning you can't build anything you can imagine anymore. Because of this problem, and because I'm too lazy to sort them myself, I decided to spend way more time than it would take to sort them by hand to build a lego sorter.

The plan:

My plan was to build a machine that could sort legos by color. I would dump a bunch of legos into a hopper, and the machine would use a conveyor belt and camera to sort the legos into different bins based on their color. I started off by doing some research to deepen my understanding of CAD (Computer Aided Design) and then designing the machine in FreeCAD, an open source CAD software. It took a lot of tries to get the design right, and many rolls of fillament were harmed in the making of this project.

Prooving the concept:

Before building anything, I also wanted to make sure that the code would actually work. I wrote a program in Python that could take a picture of a lego, and determine its color. As expected, this was pretty easy. Python's image processing libraries are very powerful, and it didn't take too long to get a program that could determine the color of a lego with a high degree of accuracy simply by counting the number of pixels of each color in the image.

Building the machine:

Once I was convinced that the project could work, I could justify obtaining the electronics, motors, and other parts I needed to build the machine. I printed as many parts and could be 3D printed, and bought the rest. Then I re-printed all the parts more times than I can count because they broke, didn't fit, or needed to do something slightly different. I'm still in the prototyping phase, but currently the conveyor belt and camera are working, and it's able to sort legos into two bins based on their color.

Next Steps:

There are a few things I'd like to do to improve the machine:

  • First, I haven't yet added the hopper, meaning I have to manually place legos on the conveyor belt. This essentially defeats the purpose of an automated lego sorter, so I'll need to add that.
  • Second, I'd like to add more bins. Currently the machine can only sort legos into two bins, but I'd like to add more so that I can sort legos by color more effectively.
  • One other thing that could be very interesting is using a more advanced program, such as a neural network to classify legos. This would allow the machine to sort legos by shape, size, or even type, not just color.

Key learnings:

  • CAD tools are very powerful, and learning to use them more effectively has opened up a lot of future project possibilities.
  • I gained a bit of confidence in my ability to design and build a machine from scratch. I've generally been more of a software person, so this was definitely out of my comfort zone, but a very valuable experience.
  • Knowing the plan is very important. My first impulse was to just start buying parts and building the machine, but the easy-to-see cost of parts on amazon made me stop and think about what I was doing. The costs associated with purely software projects are more hidden, but the concept transfers, and I this project contributed to my understanding of the importance of planning.

I'll likely come back to this project in the future, but for now I'm happy with the progress I've made.
