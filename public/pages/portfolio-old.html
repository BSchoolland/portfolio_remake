My Old Portfolio

This project was my first portfolio website. Unlike my current portfolio, this one was built using Next.js, and was focused on looking visually impressive.

Project Overview

The project aimed to showcase my work in a creative and visually appealing way. It featured an unconventional layout, with some mini interactive projects in the hero section instead of the usual image of myself. The design sported a lot of diagonal lines and and a very blue color scheme, as well as several animations.

So why did I change it?

While the visual design of this portfolio was impressive, it had some issues with mobile responsiveness, and was not as easy to update or maintain as I would have liked. The main issue was the way I had structured the project. I had focused on making it look unique, rather than doing things the right way. This could possibly work for someone more into design than development, but my focus is on the more technical aspects of coding, so I decided to rebuild it using plain HTML, CSS, and JavaScript. My new focus is on providing solid information in a way that is as easy to understand as well as update as possible, with visuals taking a back seat.
