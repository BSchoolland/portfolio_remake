Project: Wrecking Wheels

I've always wanted to make a game, and this browser game provided the perfect opportunity to live out that dream.

So, what’s the game? All my favorite games include some element of building, whether it's Minecraft or Kerbal Space Program. My favorite gameplay element is usually the ability to get creative and make something new. Unsurprisingly, my browser game, "Wrecking Wheels," aims to encourage that same creativity by confronting players with a goal, a blank canvas, and blocks to build with.

The process of making this game has included many challenges and learning opportunities. In my opinion, the best way to learn is by doing, and designing, building, and deploying this browser game forced me to do quite a bit. Here are some of the tasks I completed as part of this process:

  • Writing all the game logic in JavaScript
  • Designing a visually appealing user interface in Figma
  • Stepping out of my comfort zone to compose the game music
  • Learning to use image editing software
  • Setting up and managing a database for user accounts and progress
  • Managing a VPS using SSH
  • Setting up automated deployment using GitHub Actions

Key learnings:

  • I learned the hard way that mobile compatibility should be a top priority for web apps. I didn’t initially plan for how my game would run on mobile devices, so it took a lot of work to get it even somewhat functional on a phone, and it’s still iffy. I plan on making major changes to my code to improve mobile compatibility in the near future.
  • Design should always focus on ease of future modifications. I designed my block classes this way and even made a graphical tool for the level creation process, making it super easy to add and test new levels.
  • Using the server I set up to host the game and database really opened my eyes to how servers work. An eye-opening moment was when I finally realized that my VPS was truly just a computer, no different from the one I was using, just virtual and far away. I "knew" this before, but it took getting my hands dirty installing a new OS, setting up the system, and cloning and running the game’s repository myself to really "understand" it.

I’m excited to work on the next update and would love to hear your feedback on the project.



Benjamin Schoolland
